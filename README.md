# MyCrypto
My own simple blockchain and my own cryptocurrency named X-Coin
Requirements- Python
Django Web Framework
ReactJS
How To Run:
Install Pycharm IDE
Goto frontend folder and run npm install
Run npm start in terminal
Run project from pycharm
Goto localhost:3000
